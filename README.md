# Roller Land CT Website

A clean static website for Roller Land CT, built with HTML, CSS, and Tailwind CSS.

## File Structure

```
RollerLandCT/
├── index.html                    # Root redirect page (optional)
├── rollerlandct.com/              # Main website directory
│   ├── index.html                # Main homepage
│   ├── pricing/
│   │   └── index.html           # Pricing page
│   ├── css/
│   │   └── main.css             # Custom styles
│   └── assets/
│       ├── images/               # Image assets
│       └── media/                # Video assets
└── README.md
```

## Deployment

This is a static site ready for deployment on Vercel or any static hosting service.

### Vercel Deployment

1. Connect your repository to Vercel
2. Set the **Root Directory** to `rollerlandct.com` in Vercel settings
3. Deploy!

Alternatively, you can deploy from the root and configure Vercel to serve from `rollerlandct.com` directory.

## Technologies

- HTML5
- Tailwind CSS (via CDN)
- Vanilla JavaScript
- Custom CSS for gradients and special effects

## Features

- Fully responsive design
